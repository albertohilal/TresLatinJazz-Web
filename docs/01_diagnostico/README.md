# Diagnóstico Inicial

## Checklist para ejecutar website-analyzer

1. **Preparación del entorno:**
   - Asegúrate de tener instalado `website-analyzer`.
   - Configura el entorno virtual si es necesario.

2. **Ejecución del análisis:**
   - Ejecuta el comando principal para analizar el sitio web.
   - Revisa los resultados de SEO, performance y OG.

3. **Registro de hallazgos:**
   - Documenta los problemas encontrados.
   - Prioriza los hallazgos según su impacto.

## Áreas de análisis
- **SEO:**
  - Metaetiquetas.
  - Palabras clave.
  - Estructura de enlaces internos.

- **Performance:**
  - Tiempos de carga.
  - Optimización de imágenes.
  - Uso de caché.

- **OG (Open Graph):**
  - Etiquetas para compartir en redes sociales.
  - Previsualización de enlaces.
