# TresLatinJazz-Web

## Descripción del proyecto
Este repositorio documenta y organiza el rediseño estratégico, técnico y comunicacional del sitio web oficial del proyecto musical Tres Latin Jazz.

## Objetivo general
Rediseñar y optimizar el sitio web oficial de Tres Latin Jazz para mejorar su impacto visual, técnico y comunicacional, con un enfoque en:
- Rediseño de la página de inicio (Home).
- Optimización SEO y performance.
- Preparación para giras internacionales (Europa y China).
- Uso de website-analyzer como herramienta de diagnóstico técnico.

## Alcance
Este proyecto se centra en la planificación, análisis y documentación del rediseño del sitio web. No incluye el desarrollo del código del sitio web.
